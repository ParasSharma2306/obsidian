How ChatLume Works

ChatLume parses your chat exports locally in your browser. Nothing is uploaded to a server.

Local File Parsing

When you drag and drop a `.txt` or `.zip` file into ChatLume, the file is read using the HTML5 File API. The contents are parsed locally by JavaScript running directly in your browser. At no point is a server or external API involved.

Privacy Model

Because everything happens locally, your data is fundamentally secure. You could even disconnect your internet after loading the page, and the tool would still parse and display your entire chat perfectly.
