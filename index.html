About Obsidian

Obsidian is a privacy-first WhatsApp chat viewer. It turns exported chats into a familiar interface with search, analytics, and proper media playback while keeping everything in your browser.

Capabilities

  • Client-side Parsing: Reads WhatsApp exports locally, including multiline messages and common system entries.
  • Inline Media: Renders images, videos, audio, stickers, and downloadable documents from ZIP exports.
  • Search + Navigation: Real-time search with highlighted matches and quick result navigation.
  • Date Jump: Jump directly to a selected date from the chat menu.
  • Analytics: View totals, media usage, and per-sender share.

Notes

  • ZIP exports with attachments give the best experience because files can be matched and previewed inline.
  • Date parsing adapts to different export formats, but malformed exports can still affect jump accuracy.
  • Media that is missing from the export is still shown clearly so gaps are obvious.

Current Limitations

  • Memory Limits: Because processing happens entirely on your device, very large ZIP exports (usually >500MB) can exceed your browser's RAM limit and crash the tab. For massive histories, exporting "Without Media" is heavily recommended.
  • Very rare localized WhatsApp system phrases may still need parser updates.
  • Date jump relies on export date markers, so broken or heavily edited logs can reduce accuracy.
  • Attachments can only render inline when the referenced files are included in the ZIP export.

Community Credit

German export support and attachment-matching improvements were sharpened with feedback from u/jacckyryan on Reddit.

How it Works

Obsidian uses the browser's native File Reader API and in-memory ZIP parsing to process your export entirely on-device. When you load a file, the script reads the raw text, uses regular expressions to identify message boundaries and timestamps, and dynamically builds a chat interface in batches to keep scrolling smooth.

For ZIP files, it extracts media into your browser's temporary memory by creating "Blob URLs", allowing images and videos to display inline without ever touching a server. Because nothing is uploaded, your privacy is guaranteed. Once you close or refresh the tab, all chat data is instantly wiped from memory.
